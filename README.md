Intelligent Code Review Assistant - Setup Instructions

1. Install Python (if not installed).
2. Install Dependencies:
Open terminal/command prompt and run:
pip install -r requirements.txt

3. Run the Application:
Run the following command in the terminal:
streamlit run app.py

4. Access:
The application will open automatically in your web browser ([[usually http://localhost:8501](https://codereviewassitantp-mx6skijk3uqf6bnvmwbsew.streamlit.app/)).](https://codereviewassitantp-mx6skijk3uqf6bnvmwbsew.streamlit.app/))

Note: The model is pre-trained and included in the model/ folder. No training is required to run the app.
